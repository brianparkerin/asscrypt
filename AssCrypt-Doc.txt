### ------------- AssCrypt --------------- ###


# Explicación:

*Función de Encriptación (encrypt_file):
    - Toma un parámetro adicional output_name, que es el nombre falso para el archivo encriptado.
    - Escribe los datos encriptados con este nuevo nombre de archivo.

*Función de Desencriptación (decrypt_file):
    - Toma un parámetro adicional output_name, que es el nombre para el archivo desencriptado.
    - Escribe los datos desencriptados con este nuevo nombre de archivo.


# Uso:
    - Especifica el nombre real del archivo y la contraseña.
    - Especifica un nombre falso para el archivo encriptado con una extensión diferente.
    - Especifica un nombre deseado para el archivo desencriptado.


# Cómo Usar
    - Guarda el script en un archivo Python, por ejemplo, encrypt_decrypt_file_aes.py.
    - Coloca el archivo que deseas encriptar en el mismo directorio que el script.


# Ejecuta el script:
Guradar como: 'python encrypt_decrypt_file_aes.py'


# El script hará:
    - Encriptar el archivo original y guardarlo con el nombre falso y la extensión especificada.
    - Desencriptar el archivo falso y guardar la salida con el nombre real especificado.
    - Asegúrate de usar la misma contraseña tanto para la encriptación como para la desencriptación. 

El nombre y la extensión del archivo encriptado no revelarán la verdadera naturaleza de su contenido, añadiendo una capa extra de seguridad.
